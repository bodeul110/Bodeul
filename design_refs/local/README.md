# 로컬 디자인 참조 자산

이 디렉터리는 Git에 올리지 않는 로컬 전용 디자인 참조 자산을 둔다.

현재 기준:

- `보들 가이드.zip`
  - 최신 피그마 원본 ZIP
- `latest_figma_2026-06-19/`
  - 최신 ZIP 해제본
- `보들 가이드.png`
  - 최신 보드형 PNG
- `bodeul_original_resolution_screens.zip`
  - 이전 기준 원본 화면 묶음
- `bodeul_split_screens/`
  - 이전 기준 원본을 화면 단위로 분할한 보조 비교 세트
- `automated_income_tool_report.md`
  - 보조 산출물이며 프로젝트 기준 문서가 아님

원칙:

- 디자인 원본은 이 디렉터리에만 둔다.
- 저장소 문서에서는 이 경로를 기준으로 참조한다.
- UI polish 기준으로만 사용하고, 기능 우선순위 판단은 기능설명서를 먼저 따른다.
- 프로젝트와 무관한 로컬 보조 산출물은 문서 점검 기준에서 제외한다.
