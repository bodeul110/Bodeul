# 기능설명서 항목별 구현 체크리스트

기준일: 2026-05-22

이 문서는 [보들_플랫폼_기능설명서.pdf](보들_플랫폼_기능설명서.pdf)의 항목을 기준으로 현재 구현 상태를 `완료`, `부분 완료`, `미구현`, `후속 설계`로 잘라 정리한 체크리스트다.

상태 기준:

- `완료`: 현재 앱/웹/운영 도구 기준으로 기본 흐름이 닫혀 있음
- `부분 완료`: 흐름은 있으나 외부 연동, 실시간성, 운영 세부가 빠져 있음
- `미구현`: 문서상 요구가 있으나 현재 사용자 흐름으로 제공되지 않음
- `후속 설계`: 기획 메모 수준이며 별도 요구 확정이 더 필요함

- 08. 실제 결제/PG 연동
- 11. 실시간 매칭 분석/푸시
- 13. GPS 실시간 위치/안심 채팅
- 16. 약국 지도/복약 상세
- 19. 실제 정산/출금 규칙

### 2. 후속 설계로 남겨둔 항목

- AI 음성 정리
- OCR 복약 비교
- 건강정보 화면
- 자동 초과 정산 고도화

## 현재 판단

- 데모/내부 테스트 기준으로는 `핵심 사용자 흐름은 대부분 닫혀 있음`
- 기능설명서 기준으로는 `완료 + 부분 완료` 상태가 중심이고, 미구현보다 `실시간/외부연동/운영정책 부족`이 더 큰 gap이다
- 다음 개발 우선순위는 새 화면 추가보다 `실시간 위치/지도`, `결제`, `정산`, `약국 상세` 같은 부분 완료 항목의 실제 연동 여부를 정하는 쪽이 맞다
