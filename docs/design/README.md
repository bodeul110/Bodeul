## 원본 기준

- 기능 기준은 `../local/보들_플랫폼_기능설명서.pdf`와 [화면 개편 목표 정리](../planning/screen-restructure-target.md)를 우선한다.
- 디자인 기준은 Figma 원본 파일을 우선한다.
- `../../design_refs/local/`의 다운로드 ZIP, PDF, PNG는 임시 export/cache로만 본다.
- Figma 파일 URL 또는 file key는 [디자인 참조 정리](../../design_refs/README.md)에 등록한 뒤 사용한다.
